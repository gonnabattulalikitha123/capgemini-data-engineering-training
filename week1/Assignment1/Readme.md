Assignment Description

 Implement an end-to-end PySpark data pipeline for car sales analysis. It includes data cleaning, validation, transformation, and analytical reporting.

📌 Steps Performed
Data Cleaning:
Handled null values
Removed negative prices
Trimmed string fields
Removed invalid foreign keys
Validation:
Compared record counts before and after cleaning
Identified invalid records
Transformation:
Calculated revenue
Generated customer, brand, and city insights
Analytics:
Dealer performance analysis
SQL-based reporting (top customers, trends)
Output:
Saved results as files/tables
